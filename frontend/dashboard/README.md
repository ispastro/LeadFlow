# LeadFlow Admin Dashboard

Clean, professional admin dashboard for managing leads and conversations.

## Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:8000
```

## Features

- ✅ Dashboard overview with stats
- ✅ Lead management table
- ✅ System health monitoring
- ✅ Clean, minimal design
- ✅ Responsive layout
- ✅ Fast navigation
